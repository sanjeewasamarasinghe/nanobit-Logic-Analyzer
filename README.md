මෙය  සදහා  යොදා ගැ‌නෙන්නේ FPGA නැමැති දෘඩඅංග කොටස ‌‌වෙි.

මෙය විවිදාකාර නිෂ්පාදකයන් විසින් නිපදවා වෙිලදපොලට සපයන අතර,  මෙය භාවිතා කරන මෟදූකාංග ඉඡිනේරැවන් විසින් ඒම  කොටස තම  සේවාදායක්ගේ අවෂ්‍යතාවය
